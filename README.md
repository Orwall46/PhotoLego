### Запуск проекта
```
git clone ...
```
Install dependencies, run virtual environment
```
python -m venv .venv
```
```
cd .venv/scripts
```
```
activate
```
```
pip install -r requirements.txt
``` 
Run

```
python main.py
```

Вы можете добавить любую другую фотографию в корень проекта и изменить main_photo_path.
